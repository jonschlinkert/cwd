# cwd [![NPM version](https://badge.fury.io/js/cwd.png)](http://badge.fury.io/js/cwd)

> Get the current working directory relative to the calling module.

## Quickstart

```js
var cwd = require('cwd');
```

## Usage

```js
var pkg = require(path.join(cwd, 'package.json'));
```
